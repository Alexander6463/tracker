1. Создайте проект в pycharm с интерпретатором python3.6
2. Далее извлеките архив tracker_python_project в директорию проекта
3. Активируйте виртуальное окружение (если не активированно, в terminal у начала строки должна быть добавка (venv))
с помощью команды: source venv/bin/activate
4. Установите poetry с помощью команды: pip install poetry
5. Установите зависимости проекат с помощью команды: poetry install
6. Запустите создание эксперимента и его анализ через команду: make run_all, либо в makefile нажмите на run_all
